# WiGLE Visualizer

WiGLE Visualizer is a browser-based playback tool for WiGLE CSV exports. It parses your observations locally in the browser, replays movement over time on a map, and estimates likely network hotspot locations with confidence levels.

## Features

- Drag-and-drop CSV loading and background parsing.
- Time playback controls with variable speed and timeline scrubbing.
- Layer toggles for route, raw observations, and estimated hotspots.
- Filtering by network type (Wi-Fi, Bluetooth, Cellular), confidence, and minimum observations.
- Type-specific markers (circle, diamond, square) plus a legend and network tooltips.
